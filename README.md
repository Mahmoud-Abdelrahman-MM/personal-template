# Personal Template

**Mahmoud** is a modern, fully responsive website template designed for personal portfolios, freelancers, and individuals. It features a clean, professional layout that showcases articles, gallery, features, testimonials, team members, services, skills, events, pricing plans, videos, stats, and discount requests effectively.

## Features

- Fully responsive design (mobile, tablet, desktop)
- Clean and minimal UI/UX
- Easy navigation with a collapsible menu and mega menu
- Sections for Articles, Gallery, Features, Testimonials, Team Members, Services, Our Skills, How It Works, Events, Pricing Plans, Top Videos, Stats, and Discount
- Customizable content and styles
- Optimized for performance

## Technologies Used

- HTML5
- CSS3 (with Font Awesome icons)
- Google Fonts (Cairo)

## Installation and Usage

1. Clone or download this repository to your local machine.
2. Open `index.html` in any modern web browser to view the template.
3. Customize the content in `index.html` and styles in `css/style.css` as needed.
4. For deployment, upload the files to your web server.
